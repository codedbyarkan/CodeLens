# 🔍 CodeLens — AI Code Reviewer

A sleek, dark-themed code review tool powered by Google Gemini 3 Flash Preview.
Paste any code, choose your language and focus area, and get instant
AI-generated feedback on bugs, security, performance, and code quality.

---

## ✨ Features

- 🤖 Powered by Google Gemini 3 Flash API
- 🎨 Dark glassmorphism UI with animated ambient orbs
- 🌐 Supports 10 languages — JS, TS, Python, Java, Go, Rust, and more
- 🔍 5 review modes — Full Review, Bugs, Security, Performance, Code Style
- ⚡ Built with React + Vite + CSS Modules
- 📦 Zero backend required — runs entirely in the browser

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React 18 | UI framework |
| Vite | Build tool |
